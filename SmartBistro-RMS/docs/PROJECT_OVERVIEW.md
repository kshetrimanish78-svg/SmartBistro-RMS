# SmartBistro Project Overview

SmartBistro is a web-based Restaurant Management System that improves restaurant operations by integrating ordering, kitchen communication, inventory control, and reporting into a single digital platform.

## Problem Statement
Many restaurants still rely on paper-based orders, manual stock tracking, and disconnected sales records. This can cause order delays, communication errors, inaccurate inventory records, and limited visibility for managers.

## Proposed Solution
SmartBistro provides a centralized system where staff can create customer orders, send them directly to the kitchen, update inventory automatically, and generate basic operational reports.

## Key Stakeholders
- Restaurant managers
- Waitstaff
- Kitchen staff
- Customers
- System administrators

## Expected Benefits
- Faster order processing
- Improved kitchen communication
- Reduced human error
- Better inventory tracking
- Data-driven decision making
