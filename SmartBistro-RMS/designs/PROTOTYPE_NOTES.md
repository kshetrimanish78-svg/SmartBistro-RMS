# Prototype Notes

The SmartBistro prototype should follow usability principles by using clear navigation, readable text, consistent screen layouts, and simple workflows for restaurant staff.

## Suggested Prototype Screens
1. Login screen
2. Staff dashboard
3. Create order screen
4. Kitchen display screen
5. Inventory management screen
6. Reports dashboard

## Design Principles
- Simple and consistent navigation
- Clear buttons and labels
- Fast access to common tasks
- Minimal screen clutter
- Role-based access for different users
